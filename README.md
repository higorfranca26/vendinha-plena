# VendinhaPlena

Descrição
Projeto desktop em .NET 10 para um sistema de vendas simples

Requisitos
- .NET 10 SDK
- Visual Studio 2022/2026 ou outro IDE compatível

Instalação e execução
1. Clone o repositório ou abra a pasta do projeto no Visual Studio:
   git clone https://github.com/higorfranca26/VendinhaPlena01
2. Abra a solução/projeto VendinhaPlena.Desktop no Visual Studio.

Estrutura do repositório
- VendinhaPlena.Desktop/      -> Projeto principal
- README.md
